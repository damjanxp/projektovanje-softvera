# Sprint 2 Plan — Tours & Key Points

## Sprint Goal
Omogućiti vodiču kreiranje tura (draft), dodavanje ključnih tačaka i objavljivanje tura, uz prikaz published tura turistima.

## Sprint Backlog (US)
- US-B1 Kreiranje ture (draft)
- US-B2 Dodavanje ključnih tačaka
- US-B3 Publish ture (min 2 key points)
- US-B4 Pregled svojih tura + sortiranje
- US-B5 Pregled published tura

## Deliverable (Inkrement)
- Guide može kreirati draft turu i dodavati key points
- Publish validacije rade
- Turisti/anon vide samo published ture

## Demo Scenario
1) Login guide
2) Create tour (draft)
3) Add 2 key points
4) Publish
5) Anonymous GET published tours
