# Frontend Sprint 5 — Ratings & Problems (Plan)

## Sprint Goal
Omogućiti turistima ocenjivanje tura i prijavu problema kroz UI.

## Scope
- Rating UI
- Problem reporting
- Status problema

## Planned Backlog Items
- Rating forma (1–5)
- Obavezni komentar za ocene 1 i 2
- Prijava problema (title + description)
- Prikaz statusa problema
- (Opcionalno) Prikaz event loga problema

## Out of Scope
- Napredna analitika ocena

## Definition of Done
- Turista može oceniti turu
- Turista može prijaviti problem
- Status problema je vidljiv korisnicima
