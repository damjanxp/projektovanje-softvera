# Frontend Sprint 0 — Setup & Infrastructure (Plan)

## Sprint Goal
Postaviti osnovnu Angular aplikaciju i infrastrukturu koja će služiti kao temelj za dalji frontend razvoj.

## Scope
- Angular project setup
- Routing i globalni layout
- API konfiguracija
- Osnovni auth infrastructure (bez UI logike)

## Planned Backlog Items
- Kreiranje Angular projekta
- Osnovna folder struktura (core/shared/features)
- Konfiguracija environment fajlova
- Global layout (navbar + router-outlet)
- ApiResponse model
- AuthService (osnovni)
- JWT HTTP interceptor

## Out of Scope
- UI dizajn
- Autentifikacija korisnika
- Role-based navigacija

## Definition of Done
- Aplikacija se uspešno pokreće (`ng serve`)
- Routing funkcioniše
- Backend API je dostupan preko environment konfiguracije
